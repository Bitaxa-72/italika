Обновили подборку сезонных ингредиентов и упаковки к весеннему меню

Подготовили подборку позиций для весеннего меню: ингредиенты для десертов, упаковку для витрин и расходные материалы для ежедневной работы цеха. Этот текст имитирует контент из редактора и дальше будет заменяться реальным материалом из админки.

В подборку вошли базовые продукты для муссовых десертов, шоколадного декора, начинок и подарочных наборов. На десктопе изображение остается рядом с текстом, а сам текст аккуратно обтекает картинку.

Что добавили

Обновили складские остатки по ходовым позициям, добавили несколько вариантов упаковки с прозрачным окном и расширили выбор сезонного декора. Для оптовых клиентов позиции доступны в стандартных партиях.

  • упаковка для тортов, эклеров и порционных десертов;
  • ингредиенты для кремов, начинок и шоколадного декора;
  • расходные материалы для кондитерских цехов.
